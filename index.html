Git Commands

Description It is a distributed version controll system to co-ordinate between many developers.
Commads
Command Use
git init Initialize local git repository in the folder creating .git folder.
git status Check status of working tree.
git add It will add the file to the staging area and it will ready the changes for commit
git add . It will add the changes of all changed file to the staging area and it will ready the changes for commit
git commit It will commit the changes of the file to index and will push the changes to the local repository.
git commit -a -m "commit message" It will commit the changes of all files to index with a message and will push the changes to the local repository.
git push It will push the changes and the code from the local repository to remote repository
git pull By using this command we can pull latest changes to our local repository from the remote repository.
git clone By using git clone, we can clone a remote repository in to a new directory.

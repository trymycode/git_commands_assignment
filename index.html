Git Commands

GIT - It is a distributed version control system to co-ordinate between many developers.

Commands
Commands Use
git init Initialize local git repository in the folder creating .git folder.
git status Check status of working tree.
git add filename It will add the file to the staging area and it will ready the changes for commit
git add .filename It will add the changes of all changed file to the staging area and it will ready the changes for commit
git commit It will commit the changes of the file to index and will push the changes to the local repository.
git commit -a -m "commit message" It will commit the changes of all files to index with a message and will push the changes to the local repository.
git push It will push the changes and the code from the local repository to remote repository
git pull By using this command we can pull latest changes to our local repository from the remote repository.
git clone By using git clone, we can clone a remote repository in to a new directory.
git rm --cached filename By this command you can remove(rm stands for remove) any file from your repository.
git checkout 'branch name' By using this command you can shift from one branch to another of a project
git branch 'new branch name' By using this command you can create a new branch
git merge 'branch name' By using this command you can merge the different branch codes
git log --all By using this command you can see all the command details(command time, date, author name, changed file details)
git log -number Example-> git log -2 By using this command you can see the last 2 command details.
git log --stat We can use this command to view summary of the changes made in each commit
git log --oneline To just get the bare minimum information in a single line per commit, use the â€“oneline option
git log --author=authorname Another common way to filter commits is by the person name who wrote or committed the changes.
git log --committer=committername Another common way to filter commits is by the person name who committed the changes.
git reset --hard HEAD It will discard all changes in files since the last commit
git push origin branchname It is used to set the local branch to remote branch

Git Ignore Files

We can create .gitignore file and can include the name of the file which we don't want to add in git repository.
